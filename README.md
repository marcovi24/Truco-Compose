## 🔍 What is Truco-Compose?

Truco-Compose is an open-source Android card game app. It uses Kotlin and Jetpack Compose to build a clean and modern design. The game separates the core game rules from the user interface. This makes it easier to manage and update. The app also uses a modern way to handle app state, helping it run smoothly.

This app was made as a portfolio project. It shows how to build Android apps with clear structure and good practices.

Look for a file with the `.apk` extension. This file is the installation package for Android devices.

- Find the latest `.apk` file.

Before installing, you will need to make sure your Android phone allows app installs from outside the Google Play Store.

- Open Settings on your Android phone.
- Find "Security" or "Privacy" settings.
- Enable "Install from unknown sources" or "Allow this source" for the app you will use to install the file (like your file manager or browser).

### Step 4: Transfer the APK File to Your Android Device

You can move the `.apk` file from your PC to your phone using one of these methods:

- Use a USB cable:

  - Copy the `.apk` file into a folder on your phone, like "Downloads."
- Use cloud storage services such as Google Drive or OneDrive:
  - Upload the file from your PC.
  - Access it on your phone and download locally.
- Use Bluetooth:
  - Pair your phone and PC.
  - Send the file over Bluetooth.

### Step 5: Install the App on Your Phone

Once the `.apk` file is on your phone:

- Open a file manager app on your phone.

## 🎮 About the Game

Truco is a popular card game in Brazil with simple rules but deep strategy. This app lets you play Truco smoothly on your phone. The app’s interface uses Jetpack Compose for quick and responsive layout changes.

The game logic and user interface work separately. This approach helps avoid confusion and keeps the game running well.

Key features include:

- Clear separation between game rules and UI
- Modern state management for smooth gameplay
- Simple, clean interface designed for phones
- Open source code for anyone interested in Android development or card games

## 💻 How This Project Works

This project uses these tools and patterns:

- **Kotlin**: a programming language that is easy to read and write
- **Jetpack Compose**: a tool to build user interfaces with simple, reusable components
- **MVVM architecture**: a way to organize code so UI and logic stay separate
- **Hilt dependency injection**: a system to manage app components simply and safely
- **Android open source practices**: to make the project easy to understand and build upon

This setup shows a modern way of Android app development.
